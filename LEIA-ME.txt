SISTEMA DE ESCALA PAO/APAO - V5

NOVIDADE PRINCIPAL:
Agora o banco de dados é persistente.

Isso significa:
- Você não perde mais o cadastro de funcionários ao baixar uma versão nova.
- O banco fica salvo em:
  C:\Users\SEU_USUARIO\Sistema_Escala_PAO_APAO_Dados\escala.db

COMO USAR:
1. Extraia o ZIP.
2. Clique duas vezes em INICIAR_SISTEMA.bat.
3. O sistema abrirá no navegador.

IMPORTANTE:
Nas versões antigas, o banco ficava dentro da pasta do sistema.
A partir da V4, ele fica fora da pasta, em um local fixo do Windows.

MENU NOVO:
Banco de Dados
- mostra o caminho do banco
- cria backup manual
- abre a pasta dos dados
- lista backups disponíveis

BACKUP:
O sistema cria backup automático antes de ações críticas:
- excluir funcionário
- excluir turno
- gerar escala automática apagando escala anterior
- importar funcionários
- apagar dados

EXPORTAÇÃO:
Em Funcionários e em Importar/Exportar existe botão para baixar CSV dos funcionários.
Guarde esse arquivo como segurança extra.
